A demo for use [YJS](https://docs.yjs.dev) + [Monaco Editor](https://microsoft.github.io/monaco-editor/) + Redis to implement a collaborative editing.

## Installation

- Redis Server

```bash
$ brew install redis
```

Start Redis Server on redis://127.0.0.1:6379

## Start Server

```bash
$ yarn start
```

## Start Client

```bash
$ yarn client
```
